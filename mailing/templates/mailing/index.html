Subscribe with Google

Use OAuth to avoid "confirm your sub" email.

  • Subscribe with Google

(You can also unsubscribe with Google.)

Using a non-Google email

If your email isn't associated to a Google account, you can instead send an email with the following envelope.

To subscribe@evanchen.cc
From your-username@domain.net
Subject add: your-username@domain.net

To unsubscribe, use remove instead of add in the subject line. Alternatively, every email sent contains a user-specific unsubscribe link.

Technical details

This is a really simple Django project that just uses allauth to validate Google account ownership and then stores the email in a single model. You can read the source code on GitHub if you want to see how it works.
